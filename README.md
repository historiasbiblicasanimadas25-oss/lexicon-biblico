# 📖 Lexicon - Traductor Bíblico Hebreo

Herramienta gratuita para traductores bíblicos con análisis morfológico detallado.

## ✨ Características

- 📜 Texto hebreo masorético con vocales
- 🔍 Análisis morfosintáctico palabra por palabra
- 📊 Información gramatical (Qal, Perfecto, persona, género, número)
- 📝 Campo para escribir tu traducción

## 🚀 Uso

1. Lee el texto hebreo
2. Haz clic en las palabras para analizar
3. Escribe tu traducción
