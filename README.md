# 📖 Lexicon - Traductor Bíblico Hebreo

Herramienta gratuita para traductores bíblicos con análisis morfológico detallado.

## ✨ Características

- 📜 Texto hebreo masorético con vocales y cantilación
- 🔍 Análisis morfosintáctico palabra por palabra
- 📊 Información gramatical detallada (Qal, Perfecto, persona, género, número)
- 📝 Campo para escribir tu propia traducción
- 🎨 Interfaz limpia y fácil de usar

## 🎯 Ideal Para

- Estudiantes de hebreo bíblico
- Traductores bíblicos
- Pastores y maestros
- Cualquier persona interesada en el texto original

## 📚 Datos

- **Texto:** Westminster Leningrad Codex (WLC)
- **Morfología:** Basada en estándares académicos (ETCBC)
- **Licencia:** Proyecto educativo sin fines de lucro

## 🚀 Uso

1. Lee el texto hebreo en la columna izquierda
2. Haz clic en los botones de colores para analizar cada palabra
3. Estudia el análisis morfológico detallado
4. Escribe tu traducción en el campo derecho
